# 🔄 Unit Converter

A modern and responsive **Unit Converter Web Application** built using **HTML, CSS, and JavaScript**. This project follows the **Test-Driven Development (TDD)** approach and provides fast, accurate, and real-time unit conversions.

## 🌐 Live Demo

🔗 https://loquacious-sunshine-97a50c.netlify.app/

## 💻 GitHub Repository

🔗 https://github.com/shivam1108-06/unit-converter

---

## ✨ Features

- ✅ Real-time unit conversion
- ✅ Multiple conversion categories
- ✅ Length Converter
- ✅ Weight Converter
- ✅ Temperature Converter
- ✅ Volume Converter
- ✅ Dynamic "From" and "To" unit selection
- ✅ Conversion History (Last 10)
- ✅ Copy result to Clipboard
- ✅ Local Storage support
- ✅ Clear History
- ✅ Reset Converter
- ✅ Responsive Design
- ✅ Input Validation
- ✅ Error Handling
- ✅ Clean UI

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/shivam1108-06/unit-converter.git
```

### Open Project

```bash
cd unit-converter
```

### Install Dependencies

```bash
npm install
```

### Run Tests

```bash
npm test
```

### Start Project

Simply open:

```
index.html
```

or use VS Code Live Server.
